# Installation Instructions

How to Change Your AO3 Look - Follow these steps to create and use your own custom site skin:

Step 1: Go to Skins  
Log in to AO3. Go to your Dashboard (click your username) and select Skins from the menu.

Step 2: Start a New Skin  
Click the Create Site Skin button.

Step 3: Name It  
In the Title box, give your skin a name. It has to be something no one else has used, like 'MyCustomStyle_2026'.

Step 4: Paste Your Code  
Find the big box labeled CSS and paste your code there.

Step 5: Save and Turn On  
Click Submit at the bottom. On the next screen, click the Use button to actually turn the skin on.
