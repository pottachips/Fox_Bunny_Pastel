# Fox Bunny Pastel

## Installation Instructions

To use this project, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/pottachips/Fox_Bunny_Pastel.git
cd Fox_Bunny_Pastel
npm install
```
